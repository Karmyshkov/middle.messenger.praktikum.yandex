### Сборка и запуск

Для запуска проекта требуется NodeJS >= 16, сменить версию можно с помощью:

```bash
nvm use
```

Проект собирается в [Parcel](https://parceljs.org/):

```bash
npm run build
```

Сборка и запуск статического сервера на Express:

```bash
npm run start
```

Сборка проекта в Parcel:

```bash
npm run dev
```
